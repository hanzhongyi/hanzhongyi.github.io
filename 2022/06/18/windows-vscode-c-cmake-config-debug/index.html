windows下vscode利用cmake和visual studio 2019调试c++程序

  本教程以add，sub等简单功能为例创建工程。

0.下载插件

  • CMake
  • CMake Tools
  • c/c++

1.文件结构如下

.vscode
  • c_cpp_properties.json
include
  • add.hpp

3.cmake时编译选项

  • 1.在vscode底部cmake kit中选择Visual Studio Community 2019 Release - amd64即可
  • 2.build时Set the default build target选择ALL_BUILD即可，Set the target to launch选择编译后的可执行文件
  • 3.在程序上打上断点后，按F5即可开启调试
